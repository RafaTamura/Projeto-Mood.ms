Sobre o Mood.ms

Este projeto é uma plataforma musical que permite aos usuários cadastrar e explorar músicas de acordo com as emoções e sentimentos que estão vivenciando.

Ele foi realizado como um projeto final da Academia Atos .Net, nele foi aplicado diversos conhecimentos em criação e consumo de API, framework Entity, Sql Server como banco de dados e o framework Angular.

Foi uma jornada incrível com muito aprendizado e desafios, onde tive a oportunidade de conhecer e compartilhar experiências com nossos futuros colegas de profissão .
